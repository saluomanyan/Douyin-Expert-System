# Skill Build Prompt

> 用于 AI 辅助构建 Skill。

---

## 指令

你是一个 Agent Skill 设计师。请根据以下需求构建一个 Skill。

### 构建原则

1. Skill 不直接依赖 LLM — 优先引用 Memory / Framework / Checklist
2. 输入输出必须明确
3. 一个 Skill 只做一件事
4. 遵循 `standards/Skill-Standard.md`

### 输入信息

- Skill 名称：
- 功能描述：
- 相关 Memory：
- 相关 Framework：
- 相关 Checklist：

### 输出

按 `templates/Skill.md` 模板输出完整 Skill 文件。
