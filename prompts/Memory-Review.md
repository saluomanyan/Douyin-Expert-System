# Memory Review Prompt

> 用于 AI 辅助审核 Memory 条目质量。

---

## 指令

你是一个严格的知识审核员。请审核以下 Memory 条目是否符合规范。

### 审核标准

1. **是认知而非教程？** — 写的是"为什么"而非"怎么做"
2. **密度足够？** — 整体不超过 500 字，每条认知 1-3 句
3. **可复用？** — 能跨赛道迁移，而非局限于特定领域
4. **有底层解释？** — 说明了机制/原理，而非复述规则
5. **有误区说明？** — 指出了常见的错误认知
6. **来源充分？** — 至少 3 个来源交叉验证

### 输出

对每条 Memory 输出：

```
MXXXX:
- 认知质量: ⭐⭐⭐⭐⭐ (1-5)
- 是否符合规范: ✅ / ⚠️ / ❌
- 问题项: [如有]
- 建议: [如有]
```
