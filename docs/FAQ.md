# FAQ

---

## 这个项目和"抖音运营教程"有什么区别？

教程告诉你"怎么做"。本项目告诉你"为什么这么做"+"什么时候不该这么做"。

教程是教材，本项目是**操盘手的大脑**。

---

## 为什么 Memory 要独立于 LLM？

LLM 的知识来源混杂、版本不一致、模型之间有差异。
你今天用 DeepSeek，明天换 Claude，认知就变了。

Memory 是你亲自审核过的私有知识，永远不变（除非你自己改）。

---

## Memory 和 Skills 的关系是什么？

- **Memory**：知道什么是对的（决策认知）
- **Skills**：会做什么事（执行能力）

Skill 读取 Memory 获得决策依据，然后执行。LLM 只负责最后组织语言。

---

## 我可以直接用 ChatGPT/Claude 生成 Memory 吗？

不建议直接使用。LLM 的输出需要经过**人工审核 + 多源验证 + 蒸馏**才能进入 Memory。

本项目的 `prompts/` 目录下有辅助提取的 Prompt，但最终决策权在你手里。

---

## 这个系统只适用于风景类账号吗？

不。Memory 分两个维度：

1. **通用运营认知**（L2 战略 + L3 战术）：适用于任何赛道
2. **赛道特定认知**（在 L1 世界观中体现）：目前针对风景/人文/精神探索

未来可以横向扩展到其他赛道。

---

## Phase 1 为什么"不写任何知识"？

因为架构比内容更重要。先把房子建好，再搬家具。

错误的架构会导致后期大量的返工和碎片化。
