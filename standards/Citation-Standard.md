# Citation Standard

> 引用规范

---

## 一、引用原则

本项目中所有知识都必须标注来源。无来源的知识视为个人观点，不得进入 Memory。

---

## 二、引用格式

### 网页文章

```
作者. 《标题》. 平台, YYYY-MM-DD.
URL
```

### 视频/课程

```
讲者. 《课程/视频标题》. 平台, YYYY-MM-DD.
链接（如有）
```

### 书籍

```
作者. 《书名》. 出版社, 年份.
相关章节
```

### 官方文档

```
抖音. 《文档标题》. 抖音创作者学习中心, YYYY-MM-DD.
URL
```

### 操盘手观点

```
{操盘手名称/账号}. {观点摘要}. {平台}, YYYY-MM-DD.
{原始出处链接}
```

---

## 三、引用可信度

| 类型 | 可信度 | 说明 |
|------|--------|------|
| 抖音官方文档 | ★★★★★ | 权威来源 |
| 多人交叉验证 | ★★★★☆ | 行业共识 |
| 知名操盘手公开课 | ★★★☆☆ | 单一来源但有声誉 |
| 社区帖子 | ★★☆☆☆ | 需要验证 |
| 个人经验分享 | ★☆☆☆☆ | 仅作参考 |

---

## 四、引用数量要求

| 文档类型 | 最低引用数 | 说明 |
|----------|-----------|------|
| Memory | ≥3 | 多源交叉验证 |
| Knowledge | ≥1 | 标注来源即可 |
| Framework | 无需 | 方法论文档 |
| Checklist | 引用 Memory 编号 | 检查项关联认知来源 |
