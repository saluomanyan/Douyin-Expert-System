# Hermes Adapter

> 将 Douyin-Expert-System 接入 Hermes Agent。

---

## 状态

🚧 Phase 6 开始建设。

## 计划

Hermes Adapter 将实现：
1. 将 Memory 条目自动注入为 Hermes 的 system prompt / memory
2. 将 Skill 文件注册为 Hermes 的 Skill
3. 将 Framework 注册为可被 Skill 调用的决策模块
4. 将 Checklist 注册为发布前的自动审核流程
